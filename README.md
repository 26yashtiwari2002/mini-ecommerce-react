# Mini E-Commerce React App

This is a simple mini e-commerce application built using React functional components.

## Live Demo
https://mini-ecommerce-react-orpin.vercel.app/

## Features
- Displays products fetched from a public API
- Search products by name
- Filter products by category
- Sort products by price (low to high / high to low)
- Add products to cart
- Update cart item quantity
- Stock handling with out-of-stock state
- Displays total items and total price in cart

## Tech Stack
- React (Functional Components)
- JavaScript
- CSS
- FakeStore API

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000` in the browser
